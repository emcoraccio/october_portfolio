NPR Article Scraperclose


This website is designed for an easy way to view recent npr articles, save them, and add notes to any you wish. It scrapes the npr site using cheerio, and displays the title, basic description, and category on cards for viewing. A link to the actual article is provided, and users can filter the page to display only those articles which are "locked", "unlocked", or those with notes.

It is organized using the MVC structure and is deployed on heroku
